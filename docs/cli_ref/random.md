# Random manga

## Syntax

```shell
mangadex-dl "random"
```

With filter

```shell
mangadex-dl "random" -ft "KEY=VALUE"
```

For more information about filters, see {doc}`./filters`

## Example usage

```shell
mangadex-dl "random"
```

Random manga with oneshot tags

```shell
mangadex-dl "random" -ft "included_tags=oneshot"
```
